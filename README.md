
## API Reference

#### Registrar Usuario

- **Ruta:** `http://127.0.0.1:8080/api/registrar/usuario`
- **Descripcion** : Aqui podemos registrar un usuario


```json
{
  "nombre": "usuario",
  "correo": "usuarioprueba@ejemplo.com",
  "contrasena": "123456"
}
```
